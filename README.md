# GitHub Insights

A Next.js (App Router) application that integrates with the GitHub REST API to
display repository insights using shadcn/ui components.

---

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- TanStack Query (to be integrated)
- GitHub REST API

---

## Project Status

Initial project setup completed:

- Next.js App Router configured
- Tailwind CSS configured
- shadcn/ui installed with base components
- Prettier configured for consistent formatting

Core features are under development.

---

## Getting Started

```bash
pnpm install
pnpm dev
```
